
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts.

## 📝 Tasks

### 🛠️ Create a Hangman Game

#### Description
Create a functioning Hangman game that allows players to guess letters one at a time, reveals correct guesses, and ends when the player wins or loses. The game should randomly select words from a predefined list and track the player's progress throughout the game.

#### Requirements
Completed program should:

- Randomly select words from a predefined list of at least 5 words
- Accept letter guesses from the player via user input
- Display the current progress using the `_` character for unrevealed letters
- Track and display the number of incorrect guesses remaining
- Display all previously guessed letters (both correct and incorrect)
- Automatically end the game when the word is guessed correctly
- Automatically end the game when attempts are exhausted
- Display clear win/lose messages with the final word revealed
- Allow the player to play multiple rounds without restarting the program
